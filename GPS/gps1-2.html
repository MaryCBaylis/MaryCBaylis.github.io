What is pair programming?

Pair programming is two people working together on a piece of code with equal balance of input and control to solve a problem.

Why is pair programming important to DBC?

The people working on the code can receive and incorporate feedback sooner than otherwise. Also, it is better preparation for the typical coding experience in the workforce.
